zpGalleriffic 1.0 - A ZenPhoto/ZenPage Theme

This is the first release of this theme. Please report bugs on the forum and I will try to fix as soon as I can.

  • Galleriffic is a jQuery plugin that provides a rich, post-back free experience optimized to handle high volumes of photos while conserving bandwidth.
    • Smart image preloading after the page is loaded
    • Thumbnail navigation (with pagination)
    • jQuery.history plugin integration to support bookmark-friendly URLs per-image
    • Slideshow
    • Keyboard navigation
    • Graceful degradation when javascript is not available

Installation

  1. Upload the .zip file to your zenphoto themes directory. Using cpanel file browser or similar, extract the .zip files contents.
  2. Or, unzip the downloaded .zip file locally, and upload the zpGalleriffic folder to your themes directory in your ZenPhoto installation.

In ZenPhoto administration, goto the themes tab and activate the zpGalleriffic theme. To configure, navigate to Options>Theme to view and set the available options for zpGalleriffic.
